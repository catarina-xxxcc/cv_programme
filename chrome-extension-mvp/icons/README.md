# 扩展图标

## 需要创建的图标

请创建一个 128x128 像素的图标文件：`icon128.png`

## 图标设计建议

- **主题色**：樱花粉 (#FFB6C1)
- **图标元素**：可以包含简历、文档、或者樱花元素
- **风格**：简洁、现代、友好

## 快速创建方法

### 方法 1：使用在线工具
1. 访问 [Canva](https://www.canva.com/) 或 [Figma](https://www.figma.com/)
2. 创建 128x128 像素的画布
3. 使用樱花粉色背景 (#FFB6C1)
4. 添加简历或文档图标（白色）
5. 导出为 PNG 格式，命名为 `icon128.png`

### 方法 2：使用 Emoji
1. 访问 [Emoji to PNG](https://emoji.gg/) 或类似网站
2. 选择合适的 emoji（如 📄 或 🌸）
3. 下载为 128x128 PNG
4. 使用图片编辑工具添加樱花粉背景
5. 保存为 `icon128.png`

### 方法 3：临时占位符
如果暂时没有设计工具，可以使用纯色占位符：
1. 创建一个 128x128 的樱花粉色方块
2. 保存为 `icon128.png`
3. 后续再替换为正式图标

## 文件位置

将创建好的图标放在此目录下：
```
chrome-extension-mvp/icons/icon128.png
```
