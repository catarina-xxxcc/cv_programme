### 如果遇到问题

#### 问题 1：扩展没有变化
**原因**：没有刷新扩展

**解决**：
1. 去 `chrome://extensions/`
2. 找到扩展
3. 点击刷新按钮
4. 如果还不行，关闭并重新打开 Chrome

#### 问题 2：PDF 解析失败
**可能原因**：
- PDF 是扫描版（图片），不是文本版
- PDF 有密码保护
- PDF 格式不标准

**解决方案**：
1. 检查 PDF 是否可以复制文本（打开 PDF，尝试选中文字）
2. 如果不能复制，说明是扫描版，需要 OCR
3. 尝试重新导出 PDF
4. 或者使用"手动填写"模式

#### 问题 3：DOCX 解析失败
**可能原因**：
- DOCX 文件损坏
- 格式不标准（如 .doc 旧格式）

**解决方案**：
1. 用 Word 重新保存为 .docx
2. 或者导出为 PDF 再试
3. 或者使用"手动填写"模式

#### 问题 4：自动填充没反应
**检查清单**：
- [ ] 是否成功解析了简历？
- [ ] 当前页面是否有表单？
- [ ] 打开浏览器控制台（F12）查看错误

**解决方案**：
1. 确保先解析或保存简历
2. 确保当前页面有输入框
3. 查看控制台错误信息

#### 问题 5：库文件加载失败
**检查**：
```bash
ls -lh chrome-extension-mvp/libs/
```

**应该看到**：
```
mammoth.browser.min.js  (628 KB)
pdf.min.js              (695 KB)
pdf.worker.min.js       (2.0 MB)
```

**如果文件太小或不存在**：
运行下载脚本：
```bash
./chrome-extension-mvp/下载-pdfjs.sh
```
