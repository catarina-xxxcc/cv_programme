# 🌸 简历自动填充 Chrome 扩展 - V4 版本

## 📖 项目简介

这是一个 Chrome 浏览器扩展，可以自动解析简历并填充招聘网站的申请表单，大大提高求职效率。

### 核心功能

- ✅ **智能解析**：支持 PDF、DOCX、TXT 三种格式
- ✅ **一键填充**：自动识别表单字段并填充
- ✅ **完全离线**：不依赖外部服务，数据仅存本地
- ✅ **隐私保护**：简历数据不上传，完全本地处理

## 🎯 使用场景

- 在 BOSS直聘、拉勾网、智联招聘等网站快速投递简历
- 避免重复填写相同信息
- 提高求职效率，节省时间

## 🚀 快速开始

### 1. 安装扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `chrome-extension-mvp` 文件夹
6. 扩展安装成功！

### 2. 上传简历

1. 点击浏览器工具栏的扩展图标 🌸
2. 在"📄 上传简历"标签页
3. 上传你的 PDF、DOCX 或 TXT 格式简历
4. 点击"🔍 解析简历"
5. 等待解析完成

### 3. 自动填充

1. 打开任意招聘网站的申请表单
2. 点击扩展图标 🌸
3. 点击"✨ 自动填充"按钮
4. 表单自动填充完成！

## 🐛 故障排查

### PDF 解析失败
- 确保 PDF 是文本版（不是扫描版）
- 尝试重新导出 PDF
- 或使用"手动填写"模式

### DOCX 解析失败
- 确保是 .docx 格式（不是 .doc）
- 用 Word 重新保存
- 或导出为 PDF

### 自动填充没反应
- 确保已成功解析简历
- 确保当前页面有表单
- 打开控制台（F12）查看错误
