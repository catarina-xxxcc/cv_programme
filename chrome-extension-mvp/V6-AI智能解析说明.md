## 🚀 使用指南

### 第一步：配置 DeepSeek API Key（可选但推荐）

1. 点击扩展弹窗右上角的 ⚙️ 设置按钮
2. 在新打开的设置页面中：
   - 点击"点击这里注册获取"链接，前往 DeepSeek 官网
   - 注册账号并获取 API Key（新用户有免费额度）
   - 将 API Key 粘贴到输入框
   - 点击"测试连接"确认 API Key 有效
   - 点击"保存设置"

**注意**：
- API Key 仅存储在本地，不会上传到任何服务器
- 如果不配置 API Key，扩展会使用正则表达式解析（准确率较低）

### 第二步：上传并解析简历

1. 在扩展弹窗中，点击"上传简历"标签页
2. 选择或拖拽简历文件（支持 PDF、DOCX、TXT）
3. 点击"解析简历"按钮
4. 等待解析完成（通常 2-5 秒）
5. 查看解析结果预览
6. 如有错误，点击"编辑"按钮手动修改

### 第三步：自动填充

1. 打开招聘网站的申请表单
2. 点击扩展图标，打开弹窗
3. 点击"自动填充"按钮
4. 表单自动填充完成！

## 🤖 AI 解析原理

### 工作流程

```
用户上传简历
    ↓
提取文本（PDF/DOCX/TXT）
    ↓
检查是否配置 API Key
    ↓
┌─────────────┬─────────────┐
│  有 API Key  │  无 API Key  │
└─────────────┴─────────────┘
    ↓                ↓
调用 DeepSeek API   使用正则表达式
    ↓                ↓
解析成功？          解析完成
    ↓                ↓
  是/否              ↓
    ↓                ↓
成功 → 返回结果      ↓
失败 → 回退到正则    ↓
    ↓                ↓
    └────────────────┘
            ↓
      显示解析结果
```

### AI 提示词设计

我们精心设计了提示词，确保 AI 能够：

1. **准确提取基本信息**
   - 姓名、邮箱、电话
   - 学校、专业、学历、毕业时间

2. **智能拆分教育背景**
   - 正确处理"深圳大学-金融科技专业 2023.9-2027.6"这样的格式
   - 分离学校名、专业名、时间范围

3. **保留技能分类**
   - 识别用户自定义的分类（如"语言："、"编程技能："）
   - 保存到 `skillsDetailed` 对象中
   - 同时生成合并的 `skills` 字符串

4. **提取工作经验和自我介绍**
   - 限制长度（200字以内）
   - 保留关键信息

## 🔧 技术细节

### 关键函数

#### `parseResumeWithAI(text)`
- 主函数，负责调用 AI API
- 检查 API Key 是否配置
- 构建提示词并发送请求
- 解析 JSON 响应
- 失败时回退到正则表达式

#### `parseResumeWithRegex(text)`
- 备用解析方案
- 使用正则表达式提取信息
- 与 V5 版本相同的逻辑

### API 调用参数

```javascript
{
  model: 'deepseek-chat',
  messages: [
    { role: 'system', content: '你是一个专业的简历解析助手...' },
    { role: 'user', content: prompt }
  ],
  temperature: 0.1,  // 低温度，确保输出稳定
  max_tokens: 2000   // 足够容纳完整的 JSON 响应
}
```

### 错误处理

1. **API Key 未配置**
   - 显示提示信息
   - 自动使用正则表达式解析

2. **API 调用失败**
   - 捕获错误并记录
   - 显示友好提示
   - 自动回退到正则表达式

3. **JSON 解析失败**
   - 尝试提取 JSON（可能在代码块中）
   - 失败则抛出错误并回退

## 🐛 已知问题

1. **API 调用可能较慢**
   - 取决于网络环境
   - 通常 2-5 秒，最长可能 10 秒

2. **API Key 需要用户自行获取**
   - 需要注册 DeepSeek 账号
   - 新用户有免费额度

3. **VPN 环境可能影响 API 调用**
   - 如果 API 调用失败，会自动回退到正则解析
   - 建议在稳定网络环境下使用
