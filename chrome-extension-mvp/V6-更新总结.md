## 🧪 测试步骤

### 1. 重新加载扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 找到"小小求职拿下"扩展
4. 点击"重新加载"按钮（🔄）

### 2. 测试设置页面

1. 点击扩展图标
2. 点击右上角 ⚙️ 设置按钮
3. 确认设置页面正常打开
4. 输入测试 API Key（如果有）
5. 点击"测试连接"
6. 点击"保存设置"

### 3. 测试 AI 解析（有 API Key）

1. 准备一份简历（PDF/DOCX/TXT）
2. 点击扩展图标
3. 上传简历文件
4. 点击"解析简历"
5. 等待 2-5 秒
6. 查看解析结果
7. 确认信息准确性

### 4. 测试正则解析（无 API Key）

1. 打开设置页面
2. 清空 API Key（如果有）
3. 返回扩展弹窗
4. 上传简历文件
5. 点击"解析简历"
6. 确认使用正则表达式解析
7. 查看解析结果

### 5. 测试自动填充

1. 打开招聘网站（如 Boss直聘）
2. 进入职位申请页面
3. 点击扩展图标
4. 点击"自动填充"
5. 确认表单自动填充

### 6. 测试错误处理

1. 输入错误的 API Key
2. 尝试解析简历
3. 确认自动回退到正则解析
4. 查看错误提示信息
