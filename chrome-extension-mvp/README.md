# 🌸 简历自动填充 Chrome 扩展 MVP

一个简单的 Chrome 浏览器扩展，帮助你快速填充招聘网站的申请表单。

## ✨ 功能特点

- 📝 **简历存储**：在扩展中保存你的简历信息
- ⚡ **一键填充**：自动识别并填充招聘表单字段
- 🎨 **樱花粉主题**：清新可爱的界面设计
- 🔒 **本地存储**：所有数据保存在本地，保护隐私

## 📋 支持的字段

- 姓名
- 邮箱
- 电话
- 学校
- 专业
- 学历
- 毕业时间
- 工作经验
- 技能
- 自我介绍

## 🚀 安装步骤

### 1. 准备图标文件

在安装扩展之前，需要先创建图标文件：

1. 进入 `chrome-extension-mvp/icons/` 目录
2. 阅读 `README.md` 了解如何创建图标
3. 创建一个 128x128 像素的 PNG 图标，命名为 `icon128.png`

**临时方案**：如果暂时没有图标，可以使用任意 128x128 的 PNG 图片作为占位符。

### 2. 加载扩展到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择 `chrome-extension-mvp` 文件夹
6. 扩展安装完成！

## 📖 使用方法

### 第一步：保存简历信息

1. 点击浏览器工具栏中的扩展图标
2. 在弹出的窗口中填写你的简历信息
3. 点击"💾 保存简历"按钮

### 第二步：自动填充表单

1. 打开任意招聘网站的申请表单页面（如智联招聘、前程无忧、BOSS直聘等）
2. 点击扩展图标
3. 点击"✨ 自动填充"按钮
4. 扩展会自动识别并填充表单字段

## 🎯 工作原理

### 字段匹配逻辑

扩展通过以下方式识别表单字段：

1. **字段名称**（name 属性）
2. **字段 ID**（id 属性）
3. **占位符文本**（placeholder）
4. **关联的标签**（label）

### 支持的字段关键词

| 简历字段 | 匹配关键词 |
|---------|-----------|
| 姓名 | name, 姓名, 名字 |
| 邮箱 | email, 邮箱, 电子邮件 |
| 电话 | phone, tel, 电话, 手机 |
| 学校 | school, university, 学校, 院校 |
| 专业 | major, 专业 |
| 学历 | degree, 学历 |
| 毕业时间 | graduation, 毕业 |
| 工作经验 | experience, 工作经验, 经历 |
| 技能 | skill, 技能 |
| 自我介绍 | introduction, 自我介绍, 个人简介 |

## 🔧 技术架构

### 文件结构

```
chrome-extension-mvp/
├── manifest.json       # 扩展配置文件
├── background.js       # 后台服务脚本
├── content.js          # 内容脚本（注入到网页）
├── popup.html          # 弹窗界面
├── popup.js            # 弹窗逻辑
├── icons/              # 图标文件夹
│   └── icon128.png     # 扩展图标
└── README.md           # 说明文档
```

### 核心组件

1. **Background Script** (`background.js`)
   - 管理简历数据存储
   - 处理消息通信
   - 协调各组件交互

2. **Content Script** (`content.js`)
   - 在招聘网站页面中运行
   - 检测表单字段
   - 执行自动填充

3. **Popup** (`popup.html` + `popup.js`)
   - 用户界面
   - 简历信息输入和保存
   - 触发自动填充

## 🐛 常见问题

### Q: 为什么有些字段没有被填充？

A: 可能的原因：
- 该网站使用了特殊的表单结构
- 字段名称不在支持的关键词列表中
- 表单使用了动态加载（需要等待加载完成）

**解决方法**：可以手动填写未识别的字段，或者联系开发者添加支持。

### Q: 填充后为什么有些字段显示为空？

A: 某些网站使用了特殊的输入验证机制。尝试：
1. 手动点击一下该字段
2. 重新点击"自动填充"按钮

### Q: 如何修改已保存的简历？

A: 点击扩展图标，修改表单内容后重新点击"保存简历"即可覆盖。

### Q: 数据存储在哪里？

A: 所有数据使用 Chrome 的 `chrome.storage.local` API 存储在本地，不会上传到任何服务器。

## 🔐 隐私说明

- ✅ 所有简历数据仅存储在本地浏览器中
- ✅ 不会上传到任何服务器
- ✅ 不会收集任何用户信息
- ✅ 开源代码，可自行审查

## 📝 权限说明

扩展需要以下权限：

- **storage**：保存简历数据到本地
- **activeTab**：访问当前标签页以填充表单
- **scripting**：注入内容脚本到网页
- **host_permissions (all_urls)**：在所有招聘网站上工作

## 🚧 MVP 限制

这是一个最小可行产品（MVP），有以下限制：

- ❌ 不支持 PDF/DOCX 文件解析（需要手动输入）
- ❌ 只能存储一份简历（不支持多版本）
- ❌ 不支持投递记录追踪
- ❌ 字段匹配可能不完美（依赖关键词）
