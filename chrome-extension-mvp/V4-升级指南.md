## 🔍 故障排查

### 问题 1：扩展图标没有变化
**解决**：
- 确保点击了刷新按钮
- 检查 manifest.json 是否正确更新
- 重启 Chrome 浏览器

### 问题 2：PDF 解析失败
**可能原因**：
- PDF 是扫描版（图片），不是文本版
- PDF 有密码保护

**解决**：
- 使用文本版 PDF
- 或者切换到"手动填写"模式

### 问题 3：DOCX 解析失败
**可能原因**：
- DOCX 文件损坏
- 格式不标准

**解决**：
- 重新保存 DOCX 文件
- 或者导出为 PDF 再试

### 问题 4：库文件加载失败
**检查**：
```bash
# 确认库文件存在
ls -lh chrome-extension-mvp/libs/
```

**解决**：
- 确保 libs 目录下有 3 个文件
- 检查 manifest.json 的 web_accessible_resources 配置

## 📝 使用建议

### 推荐格式优先级
1. **PDF**（最推荐）- 格式稳定，兼容性好
2. **DOCX**（推荐）- 常用格式
3. **TXT**（备用）- 解析最准确，但需要手动转换

### 简历格式要求
- 确保简历包含：姓名、邮箱、电话（必填）
- 使用标准格式（不要用表格或复杂排版）
- 文本版 PDF 优于扫描版

## 🔄 版本切换

如果需要切换回其他版本，修改 `manifest.json`：

```json
// V1 - 仅手动填写
"default_popup": "popup.html"

// V2 - CDN 版本（需要关闭 VPN）
"default_popup": "popup-v2.html"

// V3 - TXT 离线版
"default_popup": "popup-v3-offline.html"

// V4 - 本地库版本（当前）✨
"default_popup": "popup-v4-local.html"
```
