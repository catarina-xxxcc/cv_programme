# 🔄 切换到 V2 版本（支持 PDF/DOCX 解析）

## 快速切换步骤

### 步骤 1：更新 manifest.json

打开 `chrome-extension-mvp/manifest.json`，找到这一行：

```json
"default_popup": "popup.html",
```

修改为：

```json
"default_popup": "popup-v2.html",
```

完整的 `action` 部分应该是：

```json
"action": {
  "default_popup": "popup-v2.html",
  "default_icon": {
    "128": "icons/icon128.png"
  }
}
```

### 步骤 2：重新加载扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 找到"小小求职拿下 - 简历自动填充 MVP"扩展
4. 点击"重新加载"按钮（🔄 图标）

### 步骤 3：测试新功能

1. 点击扩展图标
2. 你应该看到两个标签页：
   - 📄 上传简历
   - ✏️ 手动填写
3. 选择"📄 上传简历"
4. 上传一个 PDF 或 DOCX 简历文件
5. 点击"🔍 解析简历"
6. 等待解析完成
7. 打开招聘网站，点击"✨ 自动填充"

## 如果想切换回 V1

如果你想回到原来的手动填写版本，只需：

1. 打开 `manifest.json`
2. 将 `"default_popup": "popup-v2.html"` 改回 `"default_popup": "popup.html"`
3. 重新加载扩展

## 两个版本的区别

### V1 (popup.html) - 手动填写版本
- ✅ 简单直接
- ✅ 无需上传文件
- ✅ 完全离线工作
- ❌ 需要手动输入所有信息

### V2 (popup-v2.html) - 智能解析版本
- ✅ 自动提取简历信息
- ✅ 支持 PDF 和 DOCX
- ✅ 支持拖拽上传
- ✅ 保留手动填写功能
- ⚠️ 需要联网（加载解析库）
- ⚠️ 解析准确度取决于简历格式

## 推荐使用场景

**使用 V1 的情况：**
- 你的简历格式比较特殊
- 你想要完全离线使用
- 你只需要填写几个字段

**使用 V2 的情况：**
- 你有标准格式的 PDF/DOCX 简历
- 你想节省手动输入的时间
- 你经常需要更新简历信息

## 常见问题

### Q: 为什么解析后有些字段是空的？

A: 可能的原因：
- 简历格式不标准
- 字段名称不常见
- 文件是扫描版 PDF（图片格式）

**解决方法**：切换到"✏️ 手动填写"标签页，补充缺失的信息。

### Q: 解析速度慢怎么办？

A: 
- 首次使用需要加载解析库（pdf.js 和 mammoth.js），可能需要几秒钟
- 大文件（>2MB）解析会比较慢
- 确保网络连接正常

### Q: 可以离线使用 V2 吗？

A: 
- 当前版本需要联网加载解析库
- 如果需要离线版本，可以考虑将库文件下载到本地（需要修改代码）

### Q: 解析结果不准确怎么办？

A: 
1. 切换到"✏️ 手动填写"标签页
2. 修改不准确的字段
3. 点击"💾 保存简历"
4. 修改后的信息会覆盖解析结果

---

**提示**：两个版本可以随时切换，数据是共享的（都保存在 chrome.storage 中）。
