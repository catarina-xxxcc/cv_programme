## 🔧 技术实现

### 关键改动

**1. 移除 flex 布局**
```javascript

  <div class="preview-label">1. Chainopera:</div>
  <div class="preview-value">...</div>
</div>

// 新版本（修复）

  <div style="font-weight: bold;">1. Chainopera</div>
  <div>职位：...</div>
</div>
```

## 🎯 设计原则

1. **卡片化设计**
   - 每段经历独立卡片
   - 清晰的视觉边界
   - 易于扫描和阅读

2. **颜色区分**
   - 工作经验：樱花粉 (#FF69B4) + 浅灰背景 (#f8f9fa)
   - 技能分类：蓝色 (#2196F3) + 浅蓝背景 (#f0f8ff)
   - 保持品牌色调一致

3. **层次分明**
   - 标题加粗突出
   - 内容字号适中
   - 合理的间距和留白

4. **响应式布局**
   - 使用 block 布局
   - 自适应宽度
   - 避免内容换行
