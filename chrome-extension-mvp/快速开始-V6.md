# 🚀 快速开始 - V6 AI 智能解析版本

## 第一步：获取 DeepSeek API Key

### 1. 注册 DeepSeek 账号

访问：https://platform.deepseek.com/

- 点击右上角"注册"
- 使用邮箱或手机号注册
- 完成验证

### 2. 获取 API Key

- 登录后，点击左侧菜单"API Keys"
- 点击"创建新的 API Key"
- 复制生成的 API Key（格式：sk-xxxxxxxxxxxxxxxx）
- **重要**：妥善保管 API Key，不要泄露给他人

### 3. 充值（可选）

- 新用户通常有免费额度
- 如需充值，点击"充值"按钮
- 最低充值 ¥10（可用很久）

## 第二步：配置扩展

### 1. 加载扩展到 Chrome

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 打开右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `chrome-extension-mvp` 文件夹
6. 扩展加载成功！

### 2. 配置 API Key

1. 点击浏览器工具栏的扩展图标（樱花粉色）
2. 点击弹窗右上角的 ⚙️ 设置按钮
3. 在设置页面中：
   - 粘贴你的 DeepSeek API Key
   - 点击"测试连接"确认有效
   - 点击"保存设置"

**提示**：如果不配置 API Key，扩展仍可使用，但会使用正则表达式解析（准确率较低）。

## 第三步：使用扩展

### 方式一：上传简历自动解析（推荐）

1. 点击扩展图标，打开弹窗
2. 确保在"📄 上传简历"标签页
3. 点击上传区域或拖拽简历文件
   - 支持格式：PDF、DOCX、TXT
4. 点击"🔍 解析简历"按钮
5. 等待 2-5 秒，AI 自动解析
6. 查看解析结果预览
7. 如有错误，点击"编辑"修改

### 方式二：手动填写

1. 点击扩展图标，打开弹窗
2. 切换到"✏️ 手动填写"标签页
3. 填写所有字段
4. 点击"💾 保存简历"

### 自动填充表单

1. 打开任意招聘网站（如 Boss直聘、智联招聘等）
2. 进入职位申请页面
3. 点击扩展图标
4. 点击"✨ 自动填充"按钮
5. 表单自动填充完成！

## 常见问题

### Q1: API Key 在哪里存储？

A: API Key 仅存储在浏览器本地（chrome.storage.local），不会上传到任何服务器。

### Q2: 每次解析要花多少钱？

A: 每次解析成本约 ¥0.001 - ¥0.003（不到 1 分钱）。每天解析 10 次，每月成本约 ¥0.3。

### Q3: 解析失败怎么办？

A: 
1. 检查网络连接
2. 确认 API Key 是否正确
3. 点击设置页面的"测试连接"
4. 如果 API 不可用，扩展会自动使用正则表达式解析

### Q4: 支持哪些招聘网站？

A: 理论上支持所有招聘网站，包括：
- Boss直聘
- 智联招聘
- 前程无忧
- 拉勾网
- 猎聘网
- LinkedIn
- 等等...

### Q5: 解析不准确怎么办？

A: 
1. 确保已配置 DeepSeek API Key（AI 解析准确率 95%+）
2. 点击"编辑"按钮手动修改
3. 或切换到"手动填写"标签页直接编辑

### Q6: 可以保存多份简历吗？

A: 当前版本只支持保存一份简历。未来版本会支持多简历管理。

## 技巧与建议

### 提高解析准确率

1. **使用标准简历格式**
   - 清晰的章节标题（教育背景、工作经验、技能等）
   - 规范的日期格式（YYYY-MM）

2. **简历内容完整**
   - 包含姓名、邮箱、电话等基本信息
   - 教育背景格式：学校-专业 时间范围

3. **技能分类清晰**
   - 使用冒号分隔：`语言：中文、英语`
   - 使用冒号分隔：`编程技能：Python、JavaScript`

### 节省 API 成本

1. **同一份简历不重复解析**
   - 解析一次后，结果会自动保存
   - 下次直接使用，无需重新解析

2. **简历更新后再解析**
   - 只有简历内容变化时才重新解析

3. **使用手动填写**
   - 如果只是微调信息，直接在"手动填写"标签页编辑
