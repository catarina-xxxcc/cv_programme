# 🎉 Chrome 扩展 MVP V2 更新说明

## 新增功能

### ✨ PDF/DOCX 自动解析

现在你可以直接上传简历文件，扩展会自动提取信息！

**支持的功能：**
- 📄 **PDF 文件解析**：使用 pdf.js 提取 PDF 文本
- 📝 **DOCX 文件解析**：使用 mammoth.js 提取 DOCX 文本
- 🤖 **智能信息提取**：自动识别姓名、邮箱、电话、学校、专业、学历等
- 🌏 **中英文支持**：支持中文和英文简历格式
- 🎯 **拖拽上传**：支持点击上传和拖拽上传两种方式

## 文件说明

### 新文件

1. **popup-v2.html** - 更新后的弹窗界面
   - 添加了标签页切换（上传简历 / 手动填写）
   - 添加了文件上传区域（支持拖拽）
   - 添加了解析进度显示
   - 集成了 pdf.js 和 mammoth.js CDN

2. **popup-v2.js** - 更新后的弹窗逻辑
   - 实现了 PDF 文本提取功能
   - 实现了 DOCX 文本提取功能
   - 实现了简历信息解析功能
   - 保留了原有的手动填写功能

### 原有文件（保留）

- **popup.html** - 原始版本（仅手动填写）
- **popup.js** - 原始版本逻辑
- **background.js** - 后台脚本（无需修改）
- **content.js** - 内容脚本（无需修改）
- **manifest.json** - 扩展配置（无需修改）

## 使用方法

### 方法 1：使用新版本（推荐）

1. **更新 manifest.json**，将 popup 指向新版本：

```json
{
  "action": {
    "default_popup": "popup-v2.html",
    "default_icon": {
      "128": "icons/icon128.png"
    }
  }
}
```

2. **重新加载扩展**：
   - 打开 `chrome://extensions/`
   - 找到"简历自动填充"扩展
   - 点击"重新加载"按钮

3. **开始使用**：
   - 点击扩展图标
   - 选择"📄 上传简历"标签页
   - 上传 PDF 或 DOCX 文件
   - 点击"🔍 解析简历"
   - 等待解析完成
   - 打开招聘网站，点击"✨ 自动填充"

### 方法 2：继续使用旧版本

如果你想继续使用手动填写版本，无需修改任何文件，manifest.json 默认指向 `popup.html`。

## 解析能力

### 可以提取的信息

- ✅ **个人信息**：姓名、邮箱、电话
- ✅ **教育背景**：学校、专业、学历、毕业时间
- ✅ **工作经验**：公司、职位、工作内容（前 200 字）
- ✅ **技能**：技术技能列表
- ✅ **自我介绍**：个人简介（前 200 字）

### 支持的简历格式

**中文简历：**
- 姓名、电话、邮箱、地址
- 教育经历、工作经历、项目经历
- 专业技能、自我评价

**英文简历：**
- Name, Phone, Email, Address
- Education, Work Experience, Projects
- Skills, Summary/Objective

## 技术实现

### 依赖库（通过 CDN 加载）

1. **pdf.js** (v4.0.379)
   - 用途：提取 PDF 文件中的文本
   - CDN：`https://cdnjs.cloudflare.com/ajax/libs/pdf.js/4.0.379/pdf.min.js`

2. **mammoth.js** (v1.6.0)
   - 用途：提取 DOCX 文件中的文本
   - CDN：`https://cdnjs.cloudflare.com/ajax/libs/mammoth/1.6.0/mammoth.browser.min.js`

### 解析流程

```
1. 用户上传文件
   ↓
2. 验证文件类型（PDF/DOCX）
   ↓
3. 提取文本内容
   ↓
4. 使用正则表达式匹配字段
   ↓
5. 保存到 chrome.storage
   ↓
6. 启用自动填充功能
```

## 已知限制

1. **解析准确度**：
   - 依赖简历格式的规范性
   - 复杂布局可能影响提取效果
   - 建议使用标准简历模板

2. **文件大小**：
   - 建议文件大小 < 5MB
   - 过大的文件可能导致解析缓慢

3. **扫描版 PDF**：
   - 不支持扫描版 PDF（图片格式）
   - 需要文本可选择的 PDF

4. **网络依赖**：
   - 需要联网加载 pdf.js 和 mammoth.js
   - 首次使用时可能需要等待库加载
