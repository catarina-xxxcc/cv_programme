# 解析调试说明

## 问题

1. 学校、专业、学历没有提取到
2. 技能字段太长，需要拆分

## 调试步骤

1. 打开扩展弹出窗口
2. 右键 → "检查"
3. 切换到 Console 标签
4. 上传简历并解析
5. 查看详细日志

## 需要看的日志

```
开始智能解析简历...
原始文本长度: XXX
✓ 找到邮箱: xxx
✓ 找到电话: xxx
开始提取学校...
  找到包含学校关键词的行: XXX
  使用教育背景行: XXX
  初步匹配到: XXX
✓ 找到学校: XXX
```

## 如果没有提取到

请把控制台的完整日志截图给我，我会根据日志优化算法。

特别是这几行：
- "找到包含学校关键词的行"
- "使用教育背景行"
- "初步匹配到"

这样我就能知道哪一步出了问题。
