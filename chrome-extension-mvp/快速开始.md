# 🚀 快速开始 - V3 离线版

## 第一步：重新加载扩展

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 找到"小小求职拿下 - 简历自动填充 MVP"
4. 点击右下角的 **🔄 刷新按钮**（重新加载扩展）

## 第二步：准备测试简历

### 方法 A：使用示例简历（推荐）
1. 打开 `chrome-extension-mvp/测试简历-示例.txt`
2. 复制全部内容
3. 粘贴到记事本
4. 保存为 `test-resume.txt`

### 方法 B：使用自己的简历
1. 打开你的 PDF 或 DOCX 简历
2. 全选（Ctrl+A / Cmd+A）并复制
3. 打开记事本/文本编辑器
4. 粘贴并保存为 `.txt` 文件

## 第三步：测试扩展

### 测试解析功能
1. 点击 Chrome 工具栏的扩展图标 🌸
2. 在弹出窗口中，点击"📄 上传简历"标签页
3. 点击上传区域，选择你的 `.txt` 文件
4. 点击"🔍 解析简历"按钮
5. 等待解析完成，应该显示"✅ 解析成功！"

### 测试自动填充功能
1. 打开测试表单页面：
   - 在浏览器中打开 `chrome-extension-mvp/test-form.html`
   - 或者访问任何招聘网站的申请表单
2. 点击扩展图标 🌸
3. 点击"✨ 自动填充"按钮
4. 表单应该自动填充你的简历信息

## 第四步：手动填写模式（可选）

如果解析失败或想手动输入：
1. 点击扩展图标 🌸
2. 切换到"✏️ 手动填写"标签页
3. 填写所有必填字段（姓名、邮箱、电话）
4. 点击"💾 保存简历"
5. 然后就可以使用"✨ 自动填充"功能了

## 常见问题

### Q: 点击刷新按钮后没有变化？
A: 确保你已经保存了 `manifest.json` 的修改，然后再点击刷新。

### Q: 解析失败怎么办？
A: 
- 检查 TXT 文件是否包含姓名、邮箱、电话这三个必填信息
- 尝试使用"手动填写"模式
- 查看浏览器控制台（F12）的错误信息

### Q: 自动填充没有反应？
A: 
- 确保已经成功解析或保存了简历
- 确保当前页面有表单输入框
- 打开浏览器控制台（F12）查看是否有错误

### Q: 为什么不支持 PDF/DOCX 直接上传？
A: V3 是离线版本，不依赖外部库。如果需要直接上传 PDF/DOCX，可以：
- 关闭 VPN 后使用 V2 版本
- 或者等待后续版本（会将库打包到本地）
