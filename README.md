# 小小求职拿下 ✦

> 上传你的简历，AI 帮你分析人格、推荐岗位、诊断简历质量。

🔗 **立即使用：[https://cv-programme-git-main-catarina-xxxccs-projects.vercel.app](https://cv-programme-git-main-catarina-xxxccs-projects.vercel.app)**

---

## 怎么用？

**第一步** — 打开上方链接

**第二步** — 上传你的简历（支持 PDF 或 DOCX 格式，文件大小不超过 5MB）

**第三步** — 点击「开始解析」，等待几秒钟

**第四步** — 查看结果：
- 🧠 **MBTI 人格** — 根据你的简历推断人格类型，附人格解读
- 💼 **推荐岗位** — 结合你的技能和人格，推荐适合你的岗位方向
- 📋 **简历诊断** — 检查错别字、病句、语意冗杂，给出修改建议和评分

---

## 注意事项

- 简历内容仅用于本次 AI 分析，不会被存储
- MBTI 结果基于简历文本推断，仅供参考
- 首次加载可能需要等待约 30 秒（服务器冷启动）
